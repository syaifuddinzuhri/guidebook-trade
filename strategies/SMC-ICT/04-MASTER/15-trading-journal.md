# Modul 15 — Trading Journal

> **Level**: 🏆 MASTER | **Estimasi belajar**: Ongoing (seumur karir trading)

---

## 15.1 Mengapa Journal adalah Alat Terpenting

> "Jika kamu tidak mencatat, kamu tidak belajar. Jika kamu tidak belajar, kamu tidak berkembang."

Journal adalah **cermin** yang menunjukkan siapa kamu sebagai trader:
- Mana kebiasaan buruk yang berulang?
- Mana setup yang paling menguntungkan?
- Kapan kamu biasanya trading terbaik / terburuk?
- Apakah kamu mengikuti rules sendiri?

## 15.2 Template Journal Harian

### Pre-Trade Analysis
```
═══════════════════════════════════════
TANGGAL: [Date]
SESI: [Asia / London / NY]
MOOD AWAL (1-10): [__]
KONSENTRASI (1-10): [__]
NEWS PENTING HARI INI: [ada/tidak — sebutkan]
═══════════════════════════════════════

ANALISIS PASAR:

XAUUSD (atau pair yang ditrade):
D1 Bias: [Bullish / Bearish / Ranging]
D1 Catatan: [__________________________]

H4 Struktur: [____________________________]
H4 Zona Entry: [____________________________]

H1 Setup: [____________________________]

Plan Hari Ini:
- Zona entry: [_______________________]
- Kondisi untuk entry: [______________]
- Jika kondisi tidak terpenuhi: TIDAK TRADE
```

### Trade Log
```
─────────────────────────────────────────
TRADE #[___]

Pair: [_____] | TF Entry: [_____]
Setup Type: [OB / FVG / Sweep / CISD]
Playbook: [A / B / C / D]
Kill Zone: [London / NY / Asia]

Entry: [_______]
SL: [_______] | Risk: [___ pip / $___]
TP1: [_______] | RR: [1:___]
TP2: [_______] | RR: [1:___]
Lot Size: [_______]

Checklist (centang jika terpenuhi):
[ ] Bias HTF jelas
[ ] Zona ITF valid  
[ ] Sweep likuiditas terjadi
[ ] CISD terkonfirmasi
[ ] Kill Zone aktif
[ ] RR ≥ 1:2
[ ] Lot size sesuai (max 1-2% risk)

Screenshot: [nama file / link]
─────────────────────────────────────────
```

### Post-Trade Review
```
─────────────────────────────────────────
HASIL TRADE:

Outcome: [Win / Loss / Breakeven]
Profit/Loss: [+/- pip] [+/- $] [+/- R]

Close Price: [_______]
Alasan Close: [TP hit / SL hit / Manual close]

Apakah saya mengikuti plan? [Ya / Tidak]
Jika tidak, apa yang berbeda? [_____________]

Apakah setup valid? [Ya / Tidak]
Jika loss: apakah itu "good loss" (plan benar, hasil unlucky) 
          atau "bad loss" (plan salah / tidak ikut rules)?
[____________________________________________]

Screenshot after: [nama file / link]
─────────────────────────────────────────
```

## 15.5 Equity Curve Tracking

Buat grafik equity curve di spreadsheet:

```
Kolom:
| Trade # | Date | P/L ($) | Running Balance | % Change |
|---------|------|---------|-----------------|---------|
| 0       | -    | -       | 1000.00         | -       |
| 1       | Jan1 | +20     | 1020.00         | +2%     |
| 2       | Jan2 | -10     | 1010.00         | -0.98%  |
...
```

Buat grafik dari kolom Running Balance → kamu bisa lihat:
- Apakah akun tumbuh stabil atau bergelombang?
- Kapan periode drawdown terjadi?
- Apakah ada "bencana" yang bisa dihindari?

## 15.6 Screenshot Protocol

### Apa yang harus di-screenshot:

**Sebelum Entry:**
```
- HTF chart (D1/H4) dengan markup bias, OB/FVG
- ITF chart (H1) dengan zona entry
- LTF chart (M15) dengan rencana entry, SL, TP
```

**Saat Entry:**
```
- Chart LTF dengan garis entry, SL, TP yang sudah dipasang
- Konfirmasi candle yang menjadi trigger
```

**Setelah Close:**
```
- Chart dengan trade yang sudah selesai
- Annotasikan apa yang terjadi (hit TP? hit SL? kenapa?)
```

### Nama File:
```
Format: YYYYMMDD_PAIR_TF_SETUP_RESULT
Contoh: 20240115_XAUUSD_H1_OB-CISD_WIN
        20240116_EURUSD_M15_FVG_LOSS
```

## 15.7 Psikologi dalam Journal

Catat juga aspek psikologi — ini sering diabaikan tapi sangat penting:

```
Sebelum trade:
- Apakah saya sabar menunggu setup yang valid?
- Apakah ada tekanan untuk "harus trade"?
- Apakah kondisi fisik dan mental baik?

Selama trade:
- Apakah saya pernah mau menggeser SL?
- Apakah saya menutup terlalu cepat karena takut?
- Apakah saya biarkan profit berjalan?

Setelah trade:
- Bagaimana perasaan saya setelah win? (terlalu euphoria?)
- Bagaimana perasaan saya setelah loss? (ingin revenge?)
- Apakah saya langsung buka trade baru setelah loss?
```

---

## 15.8 Tanda-Tanda Perlu Istirahat

Catat jika mengalami ini, lalu STOP trading:

```
[ ] 3 loss berturut-turut hari ini
[ ] Sudah mencapai daily loss limit (misal -3%)
[ ] Merasa emosi dan tidak bisa objektif
[ ] Tidak bisa fokus (lelah, sakit, masalah personal)
[ ] Ingin "balas dendam" setelah loss
[ ] Mengubah rules sendiri karena takut rugi
```

**Rule**: Jika centang 2 atau lebih → TUTUP CHART, istirahat.

## 15.9 Digital Journal Tools

| Tool | Kelebihan | Kekurangan |
|------|-----------|-----------|
| **Google Sheets** | Gratis, mudah grafik, bisa akses mana saja | Perlu setup manual |
| **Notion** | Tampilan bagus, bisa embed screenshot | Butuh setup awal |
| **TraderVue** | Dedicated trading journal, auto-import | Berbayar (ada free tier) |
| **Edgewonk** | Sangat lengkap, analisis mendalam | Mahal |
| **Myfxbook** | Auto-sync dengan broker | Hanya statistik, minim insight |

**Rekomendasi pemula**: Google Sheets (gratis + fleksibel)

## 15.10 Membangun Kebiasaan Journal

```
Minggu 1-2: Paksa diri sendiri catat setiap trade
            Walau cuma 3 baris, tetap catat

Minggu 3-4: Mulai tambahkan screenshot
            Review mingguan pertama

Bulan 2-3: Review mingguan rutin
           Mulai identifikasi pattern

Bulan 3+:  Journal menjadi bagian natural dari trading
           Insight dari journal mulai tampak nyata
```

**Tips**: Set reminder di HP → "JOURNAL TIME" setelah setiap sesi trading.

---

## 15.11 Kesimpulan Modul 15

- Journal adalah satu-satunya cara untuk improve secara terukur
- Pre-trade plan, trade log, dan post-trade review = 3 komponen wajib
- Review mingguan dan bulanan untuk melihat pola
- Catat juga aspek psikologi — bukan hanya angka
- Konsistensi mencatat > sempurna tapi jarang
