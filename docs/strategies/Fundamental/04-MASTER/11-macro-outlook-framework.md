## Pendahuluan

Modul terakhir ini mengangkat perspektif Anda ke level tertinggi — melihat market seperti ekonom dan fund manager professional. Macro outlook adalah tentang membaca "big picture" kondisi ekonomi global selama beberapa bulan hingga setahun ke depan dan memposisikan diri Anda sejalan dengan arus besar tersebut.

## 1. Monetary Policy Cycle — Siklus Kebijakan Moneter

```
MONETARY POLICY CYCLE
═══════════════════════════════════════════════════════════

Setiap siklus ekonomi memiliki pola kebijakan moneter yang relatif prediktif:

FASE 1: RECOVERY (Setelah resesi)
  ──────────────────────────────────────────────────────
  Kondisi ekonomi:
  - GDP tumbuh kembali (dari negatif ke positif)
  - Unemployment masih tinggi
  - Inflasi masih rendah (di bawah target)
  - Kredit mulai mengalir lagi

  Kebijakan bank sentral:
  - Suku bunga SANGAT RENDAH (near zero)
  - QE berjalan / baru selesai
  - Language: "Accommodative for a long time"
  - Stance: VERY DOVISH

  Asset performance:
  ┌──────────────────────────────────────────────────┐
  │ Equities:  ▲▲▲ (best performer, cheap + stimulus)│
  │ Gold:      ▲▲  (real yield negatif, USD lemah)   │
  │ Bonds:     → flat (yield rendah, tidak menarik)  │
  │ USD:       ▼▼  (rate rendah, QE melemahkan)      │
  │ AUD/NZD:   ▲▲  (risk-on, commodity demand)       │
  │ JPY:       ▼   (carry trade naik lagi)            │
  └──────────────────────────────────────────────────┘

FASE 2: EXPANSION (Ekonomi Tumbuh)
  ──────────────────────────────────────────────────────
  Kondisi ekonomi:
  - GDP tumbuh kuat (2-4%+)
  - Unemployment turun mendekati full employment
  - Inflasi mulai naik mendekati target
  - Konsumsi dan investasi kuat

  Kebijakan bank sentral:
  - Mulai "Tapering" QE
  - Mulai naikkan suku bunga (sedikit-sedikit)
  - Language: "Gradual tightening"
  - Stance: Bergerak dari DOVISH ke NEUTRAL

  Asset performance:
  ┌──────────────────────────────────────────────────┐
  │ Equities:  ▲▲  (earnings naik, tapi rate naik)  │
  │ Gold:      →   (mixed: real yield naik tapi risk │
  │                 on juga = berlawanan)            │
  │ Bonds:     ▼   (yield naik = harga bond turun)  │
  │ USD:       ▲   (rate mulai naik)                │
  │ AUD/NZD:   ▲   (commodity boom)                 │
  │ JPY:       ▼   (carry trade terus)              │
  └──────────────────────────────────────────────────┘

FASE 3: OVERHEATING (Ekonomi Kepanasan)
  ──────────────────────────────────────────────────────
  Kondisi ekonomi:
  - GDP sangat kuat
  - Unemployment sangat rendah (tight labor market)
  - Inflasi MELEBIHI target (terlalu tinggi)
  - Upah naik, demand sangat kuat

  Kebijakan bank sentral:
  - Naikkan rate agresif (50-75bps per meeting)
  - QT (Quantitative Tightening) aktif
  - Language: "Committed to restoring price stability"
  - Stance: VERY HAWKISH

  Asset performance:
  ┌──────────────────────────────────────────────────┐
  │ Equities:  ▼▼  (rate tinggi = valuation turun)  │
  │ Gold:      ▼▼  (real yield tinggi = gold mahal  │
  │                 dibanding bond berbunga)         │
  │ Bonds:     ▼▼  (yield naik = harga turun)       │
  │ USD:       ▲▲▲ (rate tertinggi = currency naik) │
  │ AUD/NZD:   ▼   (global growth fear)             │
  │ JPY:       ▼▼  (divergence dengan BoJ dovish)   │
  └──────────────────────────────────────────────────┘

  CONTOH NYATA: 2022
  Fed: 0.25% → 4.5% dalam satu tahun (tercepat dalam 40 tahun!)
  DXY: 95 → 114 (+20%)
  EURUSD: 1.15 → 0.96 (-16%!)
  XAUUSD: 2000 → 1620 (-19%)
  S&P 500: 4800 → 3500 (-27%)

FASE 4: SLOWDOWN / DISINFLATION (Inflasi Mereda)
  ──────────────────────────────────────────────────────
  Kondisi ekonomi:
  - GDP mulai melambat (dari 3% ke 1-2%)
  - Unemployment mulai naik
  - Inflasi turun menuju target
  - Kredit mulai ketat

  Kebijakan bank sentral:
  - Berhenti naikkan rate (PAUSE/HOLD)
  - Diskusi tentang timing rate cut
  - Language: "Data dependent"
  - Stance: HAWKISH → NEUTRAL → mulai DOVISH

  Asset performance:
  ┌──────────────────────────────────────────────────┐
  │ Equities:  ▲   (cut expectation = multiple naik) │
  │ Gold:      ▲▲  (real yield turun, USD melemah)   │
  │ Bonds:     ▲   (yield turun = harga naik)        │
  │ USD:       ▼   (cut expectation)                 │
  │ AUD/NZD:   →   (mixed)                          │
  │ JPY:       ▲   (divergence menghilang)           │
  └──────────────────────────────────────────────────┘

  CONTOH NYATA: 2023-2024
  Fed: Hold di 5.25-5.5%, kemudian mulai cut 2024
  XAUUSD: 1800 → 2700+ (+50% dalam 2 tahun)
  EURUSD: 1.05 → 1.12 (recovery)
  US10Y: 5.0% → 3.5% (yield turun)

FASE 5: RECESSION (Resesi)
  ──────────────────────────────────────────────────────
  Kondisi ekonomi:
  - GDP negatif (2 kuartal berturut = definisi resesi)
  - Unemployment naik signifikan
  - Inflasi bisa turun drastis (atau deflasi)
  - Kredit freezes, likuiditas ketat

  Kebijakan bank sentral:
  - Cut rate agresif
  - Kemungkinan QE baru
  - Language: "Everything necessary to support economy"
  - Stance: VERY DOVISH

  Asset performance:
  ┌──────────────────────────────────────────────────┐
  │ Equities:  ▼▼▼ (earnings turun, kredit ketat)   │
  │ Gold:      ▲▲  (safe haven + rate cut)           │
  │ Bonds:     ▲▲  (flight to quality)               │
  │ USD:       Mixed (safe haven vs. rate cut)       │
  │ AUD/NZD:   ▼▼  (risk-off, demand turun)         │
  │ JPY/CHF:   ▲▲  (safe haven)                     │
  └──────────────────────────────────────────────────┘
  → Kemudian kembali ke Fase 1 (Recovery)
```

## 2. Menentukan "Regime" Market Saat Ini

```
CARA MENENTUKAN REGIME SAAT INI
═══════════════════════════════════════════════════════════

Gunakan 5 pertanyaan ini untuk menentukan fase saat ini:

1. GDP tumbuh, melambat, atau negatif?
   Naik kuat → Expansion atau Overheating
   Melambat → Slowdown atau Recession risk
   Negatif → Recession

2. Inflasi di atas, mendekati, atau di bawah target (2%)?
   Jauh di atas → Overheating
   Mendekati → Normal
   Di bawah → Recovery atau Recession

3. Unemployment: turun, stabil, atau naik?
   Turun → Expansion atau Overheating
   Stabil → Plateau
   Naik → Slowdown atau Recession

4. Bank sentral: naikkan, hold, atau turunkan rate?
   Naikkan agresif → Overheating (akhir siklus)
   Hold dengan hawkish bias → Late Expansion atau Slowdown
   Mulai bicara cut → Slowdown atau awal Recovery
   Cut agresif + QE → Recession

5. Real yield: positif dan naik, atau negatif dan turun?
   Positif dan naik → Overheating atau Tightening
   Nol / mendekati → Transition
   Negatif → Recovery atau Stimulative phase

SCORING CHART:
  ┌─────────────────────────────────────────────────┐
  │ Regime        │ GDP    │ CPI  │ Unemp │ CB     │
  ├─────────────────────────────────────────────────┤
  │ Recovery      │ Naik   │ Low  │ Turun │ Dovish │
  │ Expansion     │ Kuat   │ Mod  │ Rendah│ Neutral│
  │ Overheating   │ Kuat   │ High │ Sangat│ Hawkish│
  │               │        │      │ rendah│        │
  │ Slowdown      │ Lambat │ Turun│ Naik  │ Pause  │
  │ Recession     │ Negatif│ Low  │ Tinggi│ Very   │
  │               │        │      │       │ Dovish │
  └─────────────────────────────────────────────────┘
```

## 3. Posisi Asset di Setiap Regime

```
ASSET ALLOCATION PER REGIME
═══════════════════════════════════════════════════════════

RECOVERY REGIME:
  Best pairs: Long AUD/USD, Long EUR/JPY, Long risk currencies
  Gold: Buy the dips (real yield turun mendukung)
  Equities: Strong buy
  Avoid: Short emerging market, Short AUD/NZD

  Trade idea: Long AUD/JPY (risk-on + commodity + carry)

EXPANSION REGIME:
  Best pairs: Long USD (rate naik), Long commodity currencies
  Gold: Neutral to slight sell (yield naik)
  Equities: Moderate buy (tapi mulai hati-hati)
  Avoid: Short USD prematurely

  Trade idea: Long USD/JPY (divergence: Fed hike, BoJ hold)

OVERHEATING REGIME:
  Best pairs: Long USD aggressively, Short commodity untuk late cycle
  Gold: SELL (real yield tertinggi, opp cost tinggi)
  Equities: Short or avoid (P/E compression karena rate)
  Avoid: Long gold, Long risk currencies

  Trade idea: Short EUR/USD, Short AUD/USD, Short gold

SLOWDOWN REGIME:
  Best pairs: Long Gold, Long JPY, Long CHF
  Gold: STRONG BUY (pivot expectation, real yield turun)
  Equities: Cautious, selective
  Avoid: Long USD (cut expectation = melemah)

  Trade idea: Long Gold, Long EUR/JPY (risk-on recovery)

RECESSION REGIME:
  Best pairs: Long JPY, Long CHF, Long Gold (after initial chaos)
  Gold: Buy on dips (initial chaos: bisa turun dulu)
  Equities: Avoid or Short
  Avoid: Long AUD, NZD, equities

  Trade idea: Long USD/JPY then FLIP to short (complex)
```

## 6. Framework Pembuat Macro View

```
LANGKAH-LANGKAH BUAT MACRO VIEW
═══════════════════════════════════════════════════════════

LANGKAH 1: TENTUKAN REGIME (10 menit)
  Kumpulkan 5 data terbaru:
  - GDP growth rate
  - CPI/inflasi
  - Unemployment rate
  - Central bank rate dan stance
  - Real yield

  Cocokkan dengan tabel regime → tentukan fase

LANGKAH 2: IDENTIFIKASI NARRATIVE (10 menit)
  "Apa cerita besar yang pasar ceritakan saat ini?"
  
  Contoh narrative:
  - "Fed akan pivot → beli gold, jual USD"
  - "AI boom → beli tech, beli USD"
  - "China recovery → beli AUD, beli oil"
  - "Stagflation → beli gold, jual equities"
  
  Narrative yang dominan menentukan arah market untuk berbulan-bulan

LANGKAH 3: CARI DIVERGENCE (15 menit)
  "Di mana ada perbedaan kebijakan antar bank sentral?"
  
  Pair trading terbaik = yang memiliki divergence paling jelas:
  Fed Hawkish vs BoJ Dovish → Long USD/JPY
  ECB Cut agresif vs Fed Hold → Short EUR/USD
  RBA Hawkish vs RBNZ Cut → Long AUD/NZD

LANGKAH 4: IDENTIFIKASI WILD CARDS (5 menit)
  "Apa yang bisa mengubah narrative secara mendadak?"
  - Election (US, EU, UK)
  - Geopolitik yang eskalasi
  - Data economic yang sangat mengejutkan
  - Kebijakan bank sentral yang surprise

  → Untuk wild cards, gunakan size lebih kecil

LANGKAH 5: BUAT TRADE PLAN (20 menit)
  Untuk setiap pair dalam radar:
  - Apa bias 1-3 bulan ke depan?
  - Level teknikal mana yang ingin di-entry?
  - Apa yang harus berubah untuk mengubah bias?
```

## 8. Macro Outlook vs Reality — Mengelola Ketidakpastian

```
KETIDAKPASTIAN DALAM MACRO TRADING
═══════════════════════════════════════════════════════════

KEBENARAN YANG HARUS DITERIMA:
  1. Tidak ada yang bisa memprediksi macro dengan pasti
  2. Analisis terbaik pun bisa salah
  3. Black swan selalu ada (COVID, SVB collapse, dll)
  4. Market bisa irasional lebih lama dari yang Anda perkirakan

CARA MENGELOLA KETIDAKPASTIAN:

A. PROBABILISTIC THINKING
   Jangan berpikir "ini PASTI terjadi"
   Berpikir "ini kemungkinan 65% terjadi, 35% tidak"
   → Posisikan size sesuai probabilitas

B. SCENARIO PLANNING
   Selalu punya 2-3 skenario, bukan hanya satu
   → Jika skenario A tidak terjadi, switch ke skenario B
   → Ini disebut "being flexible in conviction"

C. REVIEW DAN ADJUST
   Jika data baru bertentangan dengan bias Anda → ADJUST
   Ego adalah musuh trader fundamental
   Lebih baik salah dan ubah pandangan
   daripada benar dan kehilangan uang

D. POSITION SIZING SEBAGAI RISK MANAGEMENT
   High conviction setup (75%+ confident): 2% risk
   Medium conviction (60-75%): 1-1.5% risk
   Low conviction / uncertain macro: 0.5% risk
   Wild card event: 0% (keluar pasar sementara)

E. WAKTU UNTUK TIDAK TRADING
   Saat macro tidak jelas (misalnya election period)
   → Kurangi size 50-70%
   → Lebih banyak cash, tunggu clarity
   "When in doubt, stay out"
```

## Checklist Macro Outlook Bulanan

**Awal Bulan:**
- [ ] Update regime berdasarkan data terbaru
- [ ] Update stance semua bank sentral
- [ ] Identifikasi narrative dominan bulan ini
- [ ] Identifikasi divergence antar bank sentral
- [ ] Buat monthly bias per pair
- [ ] Tandai event penting bulan ini

**Review Pertengahan Bulan:**
- [ ] Apakah data yang keluar sesuai ekspektasi?
- [ ] Apakah narrative masih sama?
- [ ] Perlu update bias?
- [ ] Positioning masih sesuai?

**Review Akhir Bulan:**
- [ ] Evaluasi bias: berapa % yang terbukti benar?
- [ ] Apa yang mengejutkan dan mengapa?
- [ ] Pelajaran untuk bulan depan?
- [ ] Reset untuk bulan baru

## Latihan Modul 11

### Latihan 1 — Tentukan Regime Saat Ini
Kumpulkan data terbaru dari tradingeconomics.com:
1. US GDP (kuartal terakhir)
2. US CPI (bulan terakhir)
3. US Unemployment Rate
4. Fed Rate dan stance saat ini
5. US Real Yield (US 10Y TIPS)

Berdasarkan data ini, tentukan: kita berada di regime apa? Dan apa implication untuk Gold dan EUR/USD?

### Latihan 2 — Buat Macro Outlook Bulan Ini
Gunakan template macro outlook bulanan dan isi selengkap mungkin untuk bulan saat ini. Ini akan menjadi "north star" trading Anda selama sebulan.

### Latihan 3 — Scenario Planning
Buat 3 skenario untuk bulan depan:
1. Skenario base case (50% probabilitas)
2. Skenario bull (25% probabilitas) — untuk USD atau Gold, pilih pair
3. Skenario bear (25% probabilitas)

Untuk setiap skenario: apa trade yang akan Anda ambil?
