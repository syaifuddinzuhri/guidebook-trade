# Modul 07 — SnR Multi-Timeframe

**Level:** 🔴 HIGH  
**Estimasi waktu baca:** 30–35 menit  
**Prasyarat:** Modul 01–06

---

## Tujuan Modul

Setelah membaca modul ini, kamu akan:
- Memahami mengapa HTF (Higher Timeframe) SnR lebih kuat dari LTF
- Bisa melakukan analisis Top-Down (D1 → H4 → H1)
- Mengidentifikasi konfluensi SnR dari beberapa timeframe
- Menggunakan "nested zones" untuk entry yang lebih presisi

---

## 1. Hierarki Timeframe

Dalam trading, tidak semua timeframe setara. Level SnR di timeframe yang lebih tinggi (HTF) secara konsisten lebih kuat dan lebih dihormati oleh market.

```
HIERARKI KEKUATAN SnR BERDASARKAN TIMEFRAME:

Monthly (MN)  ████████████████████████████████████████  Terkuat
Weekly (W1)   ████████████████████████████████████
Daily (D1)    ████████████████████████████
H4            ████████████████████
H1            ████████████████
M30           ████████████
M15           ████████
M5            ████
M1            ██                                        Terlemah

Implikasi:
- Level Daily dihormati oleh ribuan trader di seluruh dunia
- Level M5 mungkin hanya dilihat oleh scalper di sesi tertentu
- Ketika level Daily dan level H4 berada di area yang sama → SANGAT KUAT
```

---

## 2. Konsep Top-Down Analysis

**Top-Down Analysis** adalah cara membaca market dari timeframe besar ke kecil:

```
TOP-DOWN ANALYSIS FLOW:

D1 (Daily)    → Tentukan BIAS dan KEY LEVEL
      ↓
H4 (4-Hour)   → Konfirmasi zona trading dan context
      ↓
H1 (1-Hour)   → Tentukan area entry yang lebih presisi
      ↓
M15/M30       → Timing entry akhir (konfirmasi candle)

Prinsip:
- Jangan pernah entry hanya berdasarkan LTF
- Selalu align dengan HTF bias
- LTF digunakan untuk mempersempit entry, bukan untuk menentukan arah
```

## 3. Mengapa HTF SnR Lebih Kuat?

```
PERBANDINGAN: D1 vs M15 Level

Level dari DAILY chart (D1):

Harga
│                             ← Level D1
│══════════════════════════════════════════════ 1.1200
│         │              │               │
│    ╭────╯         ╭────╯          ╭────╯
│    │               │               │
│    │               │               │
│    ← Bounce kuat   ← Bounce kuat   ← Bounce kuat
│    (80+ pip)       (60+ pip)       (100+ pip)
│    (berlangsung    (berlangsung    (berlangsung
│     berhari-hari)   berhari-hari)   berhari-hari)

Level dari M15 chart:

Harga
│                             ← Level M15
│─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ 1.1193
│    │         │
│    ╰──╮      ╰──╮
│       │         │
│       ↑ Bounce  ↑ Bounce
│       (3-5 pip) (5 pip)

- D1 bounce: Kuat, signifikan, berlangsung lama

**Alasan teknis:** Level D1 diperhatikan oleh hedge fund, bank, dan institusi besar yang trading dengan volume raksasa. Level M15 hanya diperhatikan oleh retail trader individual.

│                                               (3 sentuhan dalam 3 bulan)

│    │
│    │

│    │                                          (4 sentuhan dalam 4 bulan)
│    ╰──────────── (dari sini uptrend dimulai)
│

Kesimpulan dari D1:
- Bias: BULLISH (uptrend dari support ke resistance)
- Key Resistance: 1.1200 (jual area untuk swing trade)
- Key Support: 1.0800 (beli area untuk swing trade)
- Harga saat ini: 1.1050 (di tengah range)
```

### Langkah 2: Analisis H4 — Context dan Sub-Level

```
LANGKAH 2: H4 CHART — zoom ke area saat ini (sekitar 1.1050)

Harga
│
1.1100 ── H4 Resistance ─────────────────────  ← Minor resistance H4
│              ↓            ↓
│             ─┤─          ─┤─
│              │            │
│         ╭───╯        ╭───╯
│         │            │

│         ↑                  ↑
│        ─┤─                ─┤─
│         │                  │

│

Kesimpulan dari H4:
- H4 sedang ranging antara 1.1020 (support) dan 1.1100 (resistance)
- Harga saat ini di 1.1050 (tengah range H4)
- Context H4: Market konsolidasi sebelum lanjut ke D1 resistance (1.1200)

Rencana:
- Jika harga turun ke H4 support (1.1020) → cari peluang BUY
- Target dari H4 support: H4 resistance (1.1100), lalu D1 resistance (1.1200)
```

## 7. Kesalahan Umum dalam Multi-TF Analysis

### Kesalahan 1: Melawan HTF Bias

```
JANGAN LAKUKAN INI:

D1 Bias: Bearish (downtrend)
H4 Setup: BUY di support H4

→ Kamu sedang melawan trend D1 yang bearish
→ Probabilitas jauh lebih rendah
→ Bahkan jika H4 support valid, harga mungkin ditembus setelah bounce kecil

LAKUKAN INI:

D1 Bias: Bearish
H4 Setup: SELL di resistance H4 (align dengan D1)
H1 Timing: Entry setelah konfirmasi bearish candle di H1

→ Semua timeframe align → Probabilitas tinggi
```

### Kesalahan 2: Terlalu Banyak Timeframe

```
JANGAN: Analisis M1, M5, M15, M30, H1, H4, D1, W1 sekaligus
→ Terlalu banyak informasi → Paralysis by analysis

LAKUKAN: Pilih 3 timeframe yang kompatibel
├── Untuk Swing Trade: D1 → H4 → H1
├── Untuk Day Trade: H4 → H1 → M15
└── Untuk Scalping: H1 → M15 → M5
```

## 8. Panduan Cepat: 3 TF Sesuai Gaya Trading

| Gaya Trading | HTF (Bias) | MTF (Context) | LTF (Entry) |
|-------------|-----------|---------------|-------------|
| Swing Trade | D1 atau W1 | H4 | H1 |
| Day Trade | D1 | H4 | M30 atau H1 |
| Intraday | H4 | H1 | M15 |
| Scalping | H1 | M15 | M5 |

---

## 9. Checklist Multi-Timeframe Analysis

**HTF (Bias):**
- [ ] Trendnya apa di D1? (Bullish / Bearish / Sideways)
- [ ] Di mana KEY LEVEL D1 yang terdekat? (Support / Resistance)
- [ ] Apakah harga mendekati HTF level? (Ya → waspada)

**MTF (Context):**
- [ ] H4 bias align dengan D1?
- [ ] Di mana minor level H4 terdekat?
- [ ] Apakah ada konfluensi antara D1 dan H4 level?

**LTF (Entry):**
- [ ] H1 support/resistance align dengan H4?
- [ ] Ada konfirmasi candle di H1?
- [ ] SL di mana? Apakah tidak terlalu jauh/dekat?
- [ ] RR minimal 1:2?

## 10. Latihan Praktis

### Latihan 1: Top-Down 3 Pair
Lakukan analisis top-down untuk 3 pair berbeda:
- XAUUSD (Forex major)
- XAUUSD (Gold)
- GBPJPY (Cross major)

Untuk setiap pair:
1. Analisis D1 → tentukan bias dan key level
2. Analisis H4 → temukan sub-level dan context
3. Analisis H1 → tentukan zona entry

### Latihan 2: Temukan Konfluensi Multi-TF
Di XAUUSD, cari 2 area di mana level dari D1 + H4 + H1 bertumpuk dalam range 20 pip.

### Latihan 3: Journal 5 Trade
Selama minggu ini, hanya ambil trade yang memiliki konfluensi minimal 2 timeframe. Catat:
- Berapa trade yang kamu ambil?
- Berapa yang profit?
- Bandingkan dengan trade tanpa konfluensi sebelumnya

---

## Ringkasan

| Konsep | Poin Utama |
|--------|------------|
| HTF > LTF | Level Daily lebih kuat dari level M15 karena dilihat lebih banyak trader |
| Top-Down | Selalu mulai dari D1 untuk bias, turun ke LTF untuk timing entry |
| Konfluensi | Level yang bertumpuk dari beberapa TF = probabilitas tinggi |
| 3 TF Rule | Pilih 3 TF yang kompatibel dengan gaya trading kamu |
| Jangan lawan HTF | Jika D1 bearish, cari setup sell di LTF (bukan buy) |
