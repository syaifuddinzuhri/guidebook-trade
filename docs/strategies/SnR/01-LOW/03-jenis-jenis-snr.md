## Jenis-jenis Support & Resistance

### 1. Horizontal Support/Resistance

Ini adalah bentuk SnR yang paling klasik dan paling banyak digunakan. Level horisontal terbentuk dari area di mana harga pernah bereaksi sebelumnya.

**Kapan digunakan:** Selalu — ini adalah fondasi utama analisis SnR.

```
HORIZONTAL SUPPORT & RESISTANCE:

Harga
│
1.3200 ════════════════════════════════════════════  ← Resistance Horizontal
│    ↓         ↓                 ↓
│   ┌┴┐       ┌┴┐               ┌┴┐
│   │▓│       │▓│               │▓│
│   └┬┘       └┬┘     ╭─────────└┬┘
│   ╰─────────╯       │          │
│                     │          │
│   ╭─────────────────╯          │
│   │                            │
1.3050 ══════════════════════════════════════════════  ← Support Horizontal
│   ↑         ↑                  ↑
│  ┌┴┐       ┌┴┐                ┌┴┐
│  │█│       │█│                │█│
│  └─┘       └─┘                └─┘

Karakteristik:
- Berbentuk garis lurus / horisontal
- Terbentuk dari HIGH atau LOW yang pernah terjadi
- Semakin banyak sentuhan = semakin kuat
- Tidak berubah seiring waktu (static)
```

**Cara menggambar:**
1. Identifikasi swing high/low yang signifikan
2. Tarik garis horizontal dari titik tersebut
3. Cek apakah level tersebut pernah dihormati sebelumnya
4. Buat zone jika ada beberapa sentuhan berdekatan

---

### 2. Trendline Support/Resistance

Trendline adalah garis diagonal yang menghubungkan minimal 2 titik high (untuk resistance trendline) atau 2 titik low (untuk support trendline).

**Kapan digunakan:** Saat market sedang trending (uptrend atau downtrend jelas).

```
UPTREND — Trendline sebagai Support Dinamis:

Harga
│                                        ╭─── HIGH 3
│                              ╭─────────╯
│                    ╭─────────╯  ↑ Bounce dari trendline
│          ╭─────────╯  ↑ Bounce dari trendline
│╭─────────╯  ↑ Bounce pertama
│╯
│   /  /  /  /  /  /  /  /  /  /   ← Trendline Support
│  /  /  /  /  /  /  /  /  /  /
│
└────────────────────────────────────────────────→ Waktu

Cara menggambar:
1. Temukan 2 LOW yang makin tinggi (Higher Low)
2. Hubungkan dengan garis diagonal
3. Panjangkan ke kanan → ini adalah support trendline uptrend

Setup trading:
- Buy saat harga menyentuh trendline dari atas
- Konfirmasi dengan candle bullish
- SL di bawah trendline (buffer 10-20 pip)
```

```
DOWNTREND — Trendline sebagai Resistance Dinamis:

Harga
│╮
│ ╰─────────╮  ↓ Rejection dari trendline
│           ╰─────────╮  ↓ Rejection dari trendline
│                     ╰─────────╮  ↓ Rejection
│                               ╰─────────╮
│   \  \  \  \  \  \  \  \  \  \   ← Trendline Resistance
│    \  \  \  \  \  \  \  \  \  \
│
└────────────────────────────────────────────────→ Waktu

Setup trading:
- Sell saat harga menyentuh trendline dari bawah
- Konfirmasi dengan candle bearish
- SL di atas trendline (buffer 10-20 pip)
```

**Penting tentang Trendline:**
- Butuh minimal 2 titik untuk menggambar, tapi 3 titik untuk validasi
- Semakin panjang trendline = semakin signifikan
- Trendline yang sudah "diuji" 3x lebih kuat dari yang baru 2x
- Breakout dari trendline sering kali menghasilkan pergerakan besar

### 3. Moving Average sebagai Dynamic SnR

Moving Average (MA) bergerak mengikuti harga, sehingga disebut "dynamic support/resistance." Dalam uptrend, MA sering berfungsi sebagai support. Dalam downtrend, MA sering berfungsi sebagai resistance.

**MA yang paling sering digunakan sebagai SnR:**
- MA 20/21 (short-term)
- MA 50 (medium-term)
- MA 200 (long-term, sangat kuat)

```
MA 200 sebagai Dynamic Support dalam Uptrend:

Harga
│
│                          ╭─── Current price
│                     ╭────╯
│               ╭─────╯  ↑ Bounce dari MA 200
│         ╭─────╯  ↑ Bounce dari MA 200
│    ╭────╯  ↑ Bounce pertama
│────╯
│
│════════════════════════════════════════════════  ← MA 200
│  (Moving Average 200 bergerak naik bersama trend)
│
│                      ↑ Gap antara harga dan MA 200
│                        menyempit saat pullback
│
Observasi:
- Dalam uptrend yang kuat, MA 200 sering jadi "lantai" harga
- Pullback ke MA 200 sering jadi peluang buy yang baik
- Bounce dari MA 200 biasanya menghasilkan rally signifikan
```

```
MA 50 sebagai Dynamic Resistance dalam Downtrend:

Harga
│════════════════════════════════════════════════  ← MA 50 (bergerak turun)
│    ↓ Rejection dari MA 50
│   ─┼──────╮
│    │       ╰──── ─┬─────╮  ↓ Rejection lagi
│   ─┘               │     ╰──── ─┬─────╮
│                   ─┘             │     ╰───
│                                 ─┘
│
Observasi:
- Dalam downtrend, harga naik → MA 50 bertindak sebagai "langit-langit"
- Setiap kali harga mendekati MA 50 dari bawah, sering terjadi rejection
- Setup: Sell saat harga mendekati MA 50, konfirmasi dengan candle bearish
```

### 4. Round Number (Psychological Level)

Level psikologis adalah angka-angka "bulat" yang memiliki daya tarik kuat karena psikologi manusia cenderung menetapkan target dan stop loss di angka-angka yang mudah diingat.

**Contoh level psikologis:**
- EURUSD: 1.0000, 1.0500, 1.1000, 1.1500
- XAUUSD: 1800, 1900, 2000, 2100
- GBPUSD: 1.2000, 1.2500, 1.3000

```
ROUND NUMBER sebagai Support/Resistance:

XAUUSD (Gold) — Level psikologis 2000

Harga
│
2050 ─────────────────────────────────────────
│
│         ╭──╮         ╭──╮
│    ╭────╯  ╰─────────╯  ╰──────────────────
│    │
2000 ████████████████████████████████████████  ← PSYCHOLOGICAL LEVEL
│    │                                          (sangat kuat!)
│    ╰────────────────────────────────────────
│
1980 ─────────────────────────────────────────

Mengapa 2000 sangat kuat?
1. Angka bulat yang mudah diingat
2. Jutaan trader memasang order di sekitar level ini
3. Media dan analis sering menyebutnya sebagai "target" atau "support"
4. Banyak algoritma trading menggunakan round number sebagai trigger
5. Barrier psikologis → trader ragu membeli di atas atau menjual di bawah

Setup Trading di Round Number:
- Jika harga mendekati dari bawah → antisipasi resistance di 2000
- Jika harga mendekati dari atas → antisipasi support di 2000
- Semakin jarang harga menyentuh level itu, semakin kuat saat akhirnya menyentuh
```

### 5. Previous High/Low (PWH/PWL)

Previous Week High/Low, Previous Day High/Low, Previous Month High/Low adalah level SnR yang sangat kuat karena menggambarkan batas tertinggi dan terendah dalam periode waktu yang sudah selesai.

**Signifikansi:**
- PWH (Previous Week High): Trader melihat ini sebagai resistance utama minggu ini
- PWL (Previous Week Low): Trader melihat ini sebagai support utama minggu ini
- PDH/PDL: Sama untuk daily

```
Previous Week High/Low sebagai SnR:

                Week A              Week B              Week C
Harga
│
│          ─────────────                             ← PWH Week A
│          │   ┌──┐    │             ↑ Harga coba
│          │ ┌─┘  └─┐  │             menembus PWH A
│          │ │      └──┘         ╭───┤ Rejection!
│    ╭─────╯ │         └────╮   ╭╯   │
│    │       │          ╭───╯───╯    │
│    │       └──────────╯            │
│          ─────────────                             ← PWL Week A
│

Cara menggunakan:
1. Di awal minggu, gambar garis di HIGH dan LOW minggu sebelumnya
2. Jika harga mendekati PWH → antisipasi resistance
3. Jika harga mendekati PWL → antisipasi support
4. Breakout dari PWH/PWL dengan volume = sinyal kuat kelanjutan trend
```

### 6. Fibonacci Retracement sebagai SnR

Fibonacci Retracement menggunakan rasio matematika untuk mengidentifikasi level potensial di mana harga mungkin berhenti saat pullback.

**Level Fibonacci utama:**
- 23.6% — Retracement dangkal (trend kuat)
- 38.2% — Retracement sedang
- **50.0%** — Bukan Fibonacci murni, tapi sangat populer
- **61.8%** — "Golden ratio" — level paling kuat
- 78.6% — Retracement dalam (trend mungkin berubah)

```
FIBONACCI RETRACEMENT dalam Uptrend:

Cara menggunakan:
1. Identifikasi swing low (A) dan swing high (B) dalam uptrend
2. Tarik Fibonacci dari A ke B
3. Level retracement adalah area potensial support

Harga
│                                B ─── 100% (swing high)
│                          ╭────╯
│                    ╭─────╯
│              ╭─────╯
│        ╭─────╯
│        │           Level Retracement:
│  A ────╯
│                         ─ ─ ─ ─ 23.6% ← support lemah
│                         ─ ─ ─ ─ 38.2% ← support sedang
│                         ─ ─ ─ ─ 50.0% ← support kuat
│                         ════════ 61.8% ← support SANGAT KUAT (Golden Ratio)
│                         ─ ─ ─ ─ 78.6% ← support terakhir
│  A ─────────────────────────────100%  ← jika tembus, uptrend invalid
│
└────────────────────────────────────────────────→ Waktu
```

```
STUDI KASUS: EURUSD — Fibonacci Bounce di 61.8%

Swing: Low di 1.0500 (A) → High di 1.1200 (B)
Range: 700 pip

Level Fibonacci:
│
1.1200 ════ B (100%) ════════════════════════════
│                 ↓ Harga mulai pullback
│
1.1035 ─── 23.6% ───────────────────────────────  (harga lewat)
│
1.0932 ─── 38.2% ───────────────────────────────  (harga lewat)
│
1.0850 ─── 50.0% ─────────────────────────────── (harga lewat)
│
1.0767 ════ 61.8% ═══════════════════════════════ ← BOUNCE TERJADI DI SINI
│               │ ↑ Hammer candle muncul
│               │ ↑ Volume spike
│               ╰──────────────────── Harga rally kembali ke atas
│
1.0651 ─── 78.6% ───────────────────────────────  (tidak sampai)
│
1.0500 ─── 100% ────────────────────────────────
│

Setup:
Entry: 1.0780 (di atas level 61.8%, setelah konfirmasi hammer)
SL: 1.0720 (di bawah 61.8% + buffer)
TP: 1.1200 (kembali ke swing high)
RR: 1:7 (60 pip risk, 420 pip target)
```

## Konfluensi: Kunci Menemukan Level Terkuat

Level SnR yang paling kuat adalah ketika **beberapa jenis SnR bertumpuk di area yang sama.** Ini disebut **konfluensi**.

```
KONFLUENSI: Multiple SnR di Level yang Sama

Harga
│
1.1200 ════════════════════════════════════════════
│    ↑ Horizontal Resistance (3 sentuhan historis)
│    ↑ Fibonacci 61.8% dari swing besar
│    ↑ Round Number (1.1200 tepat)
│    ↑ Previous Week High
│
│    → Ini adalah level SANGAT KUAT dengan 4 konfluensi
│    → Probabilitas reaction di sini sangat tinggi
│    → Risk terkecil, reward potensial terbesar
│
Contoh Konkret:

XAUUSD H4 — Setup di Zona Konfluensi

2050 ─────────────────────────────────────────────  ← Batas atas zone
│    ┌──────────────────────────────────────────┐
2045 │  ZONA KONFLUENSI SANGAT KUAT              │  ← Round Number 2050
2040 │  • Horizontal Resistance (4 sentuhan)     │  ← Fibonacci 61.8%
2035 │  • Round Number 2050                      │  ← Previous Week High
2030 │  • Fibonacci 61.8%                        │  ← MA 200
     └──────────────────────────────────────────┘
2025 ─────────────────────────────────────────────  ← Batas bawah zone
│
Rekomendasi:
- Jika semua 4 konfluensi bertumpuk → ini adalah "sniper entry"
- Probability reversal sangat tinggi
- Position size bisa lebih besar (tetap dalam batas risk management)
```

## Perbandingan Semua Jenis SnR

| Jenis SnR | Sifat | Kekuatan | Terbaik Di |
|-----------|-------|----------|-----------|
| Horizontal | Static | ★★★★★ | Semua kondisi market |
| Trendline | Dynamic | ★★★★☆ | Trending market |
| Moving Average | Dynamic | ★★★☆☆ | Trending market |
| Round Number | Static | ★★★★☆ | Semua kondisi, terutama Major pairs |
| PWH/PWL | Static | ★★★★☆ | Awal minggu/bulan |
| Fibonacci | Static | ★★★★☆ | Setelah swing besar |

## Latihan Praktis

### Latihan 1: Temukan Semua Jenis SnR
Buka EURUSD Daily chart:
1. Gambar 3 horizontal support/resistance
2. Gambar trendline jika ada trend yang jelas
3. Tambahkan MA 50 dan MA 200
4. Tandai level round number terdekat
5. Identifikasi PWH dan PWL

### Latihan 2: Fibonacci Exercise
Pilih swing besar di XAUUSD (Gold):
1. Identifikasi swing low dan swing high
2. Tarik Fibonacci retracement
3. Tandai level 38.2%, 50%, 61.8%
4. Cek: apakah harga pernah bounce di salah satu level itu di masa lalu?

### Latihan 3: Cari Konfluensi
Di chart yang sama, cari area di mana minimal 3 jenis SnR bertumpuk. Tandai area itu sebagai "High Probability Zone" dan catat alasannya.

## Ringkasan

| Jenis SnR | Cara Menggambar | Kekuatan Terbesar |
|-----------|----------------|-------------------|
| Horizontal | Garis dari swing high/low berulang | Banyak sentuhan + HTF |
| Trendline | Garis diagonal dari 2+ titik | 3+ sentuhan + trend kuat |
| Moving Average | Indicator built-in | MA 200, trending market |
| Round Number | Angka bulat di chart | Pair major, angka ratusan/ribuan |
| PWH/PWL | High/Low periode sebelumnya | Awal periode baru |
| Fibonacci | Tool retracement dari swing | Level 61.8% dalam trend |
